The goal of feature engineering is to make the data better suited to the problem at hand. A simple example to consider is the "apparent temperature," which combines concrete and observable features such as air temperature, humidity, and wind speed to create a new measurement.

Feature engineering can be used to:
- Improve a model's predictive performance
- Reduce computational or data needs
- Improve interpretability of the results

---
For a feature to be useful, it has to be related to the target the model is able to learn. Any transformation applied to a feature becomes a part of the model itself. Whatever relationships the model can't learn, feature engineering can provide them through transformations.
