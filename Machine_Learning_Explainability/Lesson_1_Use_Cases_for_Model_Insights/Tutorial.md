Some insights that can be extracted from ML models include:
- What features in the data did the model think are most important?
- For any single prediction from a model, how did each feature in the data affect that prediction?
- How does each feature affect the model's predictions generally?

These insights have many uses, including:
- Debugging: Helps identify when the patterns are at odds with real world knowledge to track down bugs
- Informing Feature Engineering: Not all feature importances will be clear from real-world understandings. It also can help discover new pairings of features.
- Directing Future Data Collection: Find whether different data sources will be worthwhile or not.
- Informing Human Decision-Making: Insights can be more valuable than predictions.
- Building Trust: Understanding how the model works increases trust in it.
